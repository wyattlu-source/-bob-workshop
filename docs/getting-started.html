Getting Started

不論你是要參加 Workshop 的學員，還是想貢獻新產業案例的講師，都可以從這裡開始。

給學員

  1. 安裝並登入 IBM Bob。
  2. Clone 這個 repository 到本機。
  3. 到 Workshop 頁面選一個案例（目前只有 Medical）， 打開對應 workshops/<id>/ 資料夾。
  4. 依序讀 01 故事背景 → 02 商業需求 → 03 Prompt 設計， 打開 starter/<id>/ 的起始專案，跟著 Prompt 用 Bob 開發。
  5. 卡關時看 04 Bob 操作技巧；完成後對照 05 預期成果 與 solution/<id>/。
  6. 讀 06 重點說明，再試著做 07 挑戰題。
  7. 最後用 08 課後反思 的問題整理自己走過的 AI Coding Workflow。

給講師 / 貢獻者

想新增一個新產業的 Workshop（例如 Banking、Manufacturing、Retail）， 請參考 Repository 中的 framework/CONTRIBUTING.md。

核心概念：

  • 複製 framework/workshop-template/ 到 workshops/<id>/，依固定的 8 個章節填寫內容。
  • 在 starter/、solution/、prompt-library/、instructor/ 底下新增以 <id> 命名的資料夾。
  • 在 docs/assets/data/workshops.json 註冊 metadata，網站的 Workshop 列表會自動顯示， 不需要修改任何網頁。
