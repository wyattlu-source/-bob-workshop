IBM Bob AI Coding Workshop Framework

一套可長期維護、持續擴充的教學框架，用來快速建立不同產業的 IBM Bob 實作課程。 從需求分析、Prompt 設計、產生程式碼，到檢查、重構、完成專案，教的是完整的 AI Coding Workflow。

開始使用 瀏覽 Workshop

不是一份教材

而是一套標準化架構：故事背景、商業需求、Prompt、Bob 操作技巧、預期成果、重點說明、挑戰題、課後反思。

跨產業可擴充

Medical 是第一個範例，新增 Banking、Manufacturing、Retail 等產業不需要重新設計網站或教學流程。

教完整的工作流程

從需求分析到 Prompt 設計、產生程式碼、檢查修改、重構，學員理解的是企業如何把 AI 融入開發流程。
