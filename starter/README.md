# Starter

各 Workshop 的**課程起始程式碼**，依 workshop id 分子資料夾，例如 `starter/medical/`。

學員在課程開始時會拿到這裡的程式碼作為起點，跟著 Workshop 內的 Prompt 逐步用 IBM Bob 開發，最終應該產出對應到 [`solution/`](../solution/) 的結果。

技術棧維持 Framework 第一版原則：HTML + Bootstrap 5 + Vanilla JavaScript。
